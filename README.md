# 📊 Big Data Analytics Project

This project demonstrates the application of **Big Data Analytics techniques** using PySpark to analyze YouTube channel information. The objective is to process, clean, and analyze large datasets efficiently and extract meaningful insights.

---

## 🚀 Project Overview

In this project, we:

- Loaded a large YouTube channel dataset (CSV format)
- Performed data cleaning and preprocessing
- Applied distributed data processing using PySpark
- Conducted exploratory data analysis (EDA)
- Generated insights and visual reports

This project highlights how big data tools can be used to handle and analyze real-world datasets effectively.

## ⚙️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/Dasaram-varsha/Big-Data-Analytics-Project.git

2. Navigate to the project folder:
   cd Big-Data-Analytics-Project

3. Install required libraries:
   pip install pyspark pandas

4. Open the notebook:
   jupyter notebook Mini_Project/Mini_project.ipynb
